Jak działa API?

Interfejs REST API służy do umieszczania danych pomiarowych na serwerze przy użyciu zapytań HTTP.

Adres API to: /aplikacja/api. Metoda do przesyłania danych to: POST.

Do API należy przesłać ciąg JSON.

Ciąg JSON przyjmuje następujące klucze:

  • data - data w formacie tekstowym np.: "1993-03-12".
  • miejsce - numer id miejsca, numery id miejsc można sprawdzić w panelu administracyjnym.
  • pacjent - numer id pacjenta, numery id pacjentów można sprawdzić w panelu administracyjnym.
  • typ_pomiaru - typ pomiaru w formie tekstowej np.: "Temperatura".
  • wartosc - wartość pomiaru w formie liczby rzeczywistej.

    • Przykładowy ciąg JSON przedstawia się następująco:
    {"data": "1993-03-12", "miejsce": 1, "pacjent": 2, "typ_pomiaru": "Temperatura", "wartosc": 36.6}

    Serwer zwraca kod 200 jeżeli zapytanie się udało, oraz kod 201 jeżeli wystąpiły błędy.
